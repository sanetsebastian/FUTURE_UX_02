# FUTURE_UX_02
Aura Salon Appointment Booking App – UI/UX Design
Project Overview

This project was created as part of the Future Interns UI/UX Design Internship – Task 2.

The objective was to design a mobile-first appointment booking application for a real local business. I selected Aura Salon and designed a simple, user-friendly booking experience that allows customers to schedule salon appointments easily.

Business Chosen

Aura Salon

A salon appointment booking application designed to help customers book services quickly and efficiently without relying on phone calls or manual scheduling.

Problem Statement

Many salons still manage appointments through phone calls, messages, or manual records. This can lead to:

Missed appointments
Scheduling conflicts
Poor customer experience
Time-consuming booking processes
Solution

A mobile application that enables users to:

Browse salon services
Select their preferred stylist
Choose a convenient date and time
Confirm appointments instantly

The design focuses on simplicity, accessibility, and a smooth user experience.

Target Users
Salon customers
Working professionals
First-time visitors
Users who prefer online booking
Screens Designed
1. Welcome / Sign-Up Screen

Allows users to start their booking journey.

2. Salon Information Screen

Provides information about Aura Salon and its services.

3. Services Screen

Displays available salon services such as:

Hair Colouring
Hair Cutting
Makeup
Facial
Nail Art
Threading
4. Appointment Scheduling Screen

Users can select:

Date
Time Slot
5. Stylist Selection Screen

Users can choose their preferred stylist based on the selected service.

6. Appointment Confirmation Screen

Displays:

Selected service
Stylist name
Date and time
Booking confirmation message
7. Book Another Appointment Option

Allows users to restart the booking process easily.

User Flow

Sign Up
↓
Salon Information
↓
Services Selection
↓
Date & Time Selection
↓
Stylist Selection
↓
Appointment Confirmation
↓
Book Another Appointment

Design Decisions
Mobile-first approach
Simple and clean interface
Consistent color palette
Large touch-friendly buttons
Easy navigation between screens
Minimal booking steps to reduce user effort
